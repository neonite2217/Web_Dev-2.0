# Random Meal Generator

## üõ†Ô∏è Description

The **Random Meal Generator** is a fun and interactive website that provides users with a random meal idea every time they click the button. The app fetches meal information from a public API and displays:

- **Meal Name**: The name of the randomly generated dish.
- **Image**: A picture of the meal to help you visualize it.
- **Description**: A short description of the dish.
- **Required Ingredients**: A list of ingredients needed to prepare the meal.
- **YouTube Video**: A link to a YouTube video that shows how to prepare the dish, giving users an interactive and engaging cooking experience.

Perfect for those who want inspiration for their next meal or are looking to try something new!

## ‚öôÔ∏è Technologies Used

This project is built with the following technologies:

- **HTML5** for structuring the webpage.
- **CSS3** for styling and layout.
- **JavaScript** for handling the API request and dynamic content rendering.
- **Meal API**  [TheMealDB API](https://www.themealdb.com/api.php) to fetch random meal data, including images, descriptions, ingredients, and YouTube videos.

## üåü Features

- **Random Meal Generation**: Click the "Generate Meal" button to receive a random meal suggestion.
- **Meal Image**: Displays a visual representation of the randomly selected meal.
- **Ingredients List**: Shows the ingredients required for the recipe.
- **YouTube Video**: Links to a relevant YouTube video tutorial for cooking the dish.
- **Responsive Design**: The app is fully responsive, ensuring a great experience across devices (desktop and mobile).
